# Conference firewall

Setup a firewall that blocks all incoming requests, except those as a response on connections initiated by you.

## Setup

Copy or clone the repo into a folder.

## Start the firewall

    sudo ./iptables-load

## Reset the firewal

    sudo ./iptables-reset
